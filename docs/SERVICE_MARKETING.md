# VosCMS 홈페이지 제작 서비스

## 커스터마이즈 대응. 독자적인 예약·회원·결제 기능을 갖춘 비즈니스 사이트를 구축합니다

---

## 서비스 개요

"홈페이지를 만들고 싶지만 어디서부터 시작해야 할지 모르겠다"
"예약 시스템이 필요한데 기존 서비스는 월 비용이 부담된다"
"디자인도 기능도 내 사업에 맞게 커스터마이즈하고 싶다"

이런 고민을 가진 사업자분들을 위해, **VosCMS 기반의 올인원 비즈니스 사이트**를 구축해 드립니다.

VosCMS는 자체 개발한 차세대 CMS로, 예약·회원관리·결제·다국어(13개국어)를 기본 탑재하고 있어 별도 플러그인이나 외부 서비스 없이 **하나의 시스템으로 모든 것을 해결**할 수 있습니다.

구축 후에는 관리자 화면에서 직접 콘텐츠를 수정할 수 있어, **운영 비용을 최소화**하면서 본업에 집중할 수 있는 환경을 제공합니다.

## 플랜별 포함 기능

### 베이직 (소규모 매장·개인)

- 반응형 웹 디자인 (PC / 태블릿 / 모바일)
- 위젯 빌더로 자유로운 페이지 구성
- 온라인 예약 시스템 (서비스·스태프·일정 관리)
- 회원 등록 / 로그인 / 마이페이지
- 게시판 시스템 (공지사항, Q&A, FAQ)
- 13개국어 다국어 지원
- SEO 기본 설정
- 관리자 대시보드

### 스탠다드 (중규모·멀티 브랜치)

- 베이직 기능 전체 포함
- 멀티 브랜치(지점) 관리
- 스태프 소개 페이지 + 포트폴리오
- 고객 리뷰(口コミ) 시스템
- 포인트/쿠폰 시스템
- 이메일 알림 / LINE 알림 연동
- 결제 시스템 연동
- PWA (앱처럼 사용 가능)

### 프리미엄 (그룹·프랜차이즈)

- 스탠다드 기능 전체 포함
- 그룹 통합 관리 대시보드
- 브랜드별 독립 사이트 운영
- 키오스크 / 모바일 체크인
- 선불제(프리페이드) 시스템
- 커스텀 기능 개발
- API 연동 (외부 시스템)
- 전담 기술 지원

## VosCMS를 선택하는 이유

| 항목 | 일반 CMS (WordPress 등) | VosCMS |
|------|------------------------|--------|
| 예약 시스템 | 별도 플러그인 필요 (월 과금) | 기본 탑재 |
| 다국어 | 번역 플러그인 (품질 불안정) | 13개국어 네이티브 지원 |
| 회원 관리 | 제한적 | 풀기능 (등급, 포인트, 이력) |
| 결제 연동 | 외부 서비스 의존 | Stripe 등 직접 연동 |
| 보안 | 플러그인 충돌 위험 | 자체 개발, 일관된 보안 |
| 커스텀 | 테마/플러그인 제약 | 자유로운 커스텀 개발 |
| 운영 비용 | 플러그인 월 비용 누적 | 추가 비용 없음 |

## 기술 스택

- **CMS**: VosCMS (자체 개발)
- **언어**: PHP 8.x
- **프론트엔드**: Tailwind CSS, Vanilla JS
- **데이터베이스**: MySQL / MariaDB
- **서버**: Linux (Apache / Nginx)
- **결제**: Stripe, PayPay 등
- **PWA**: 오프라인 대응, 앱 설치 가능
- **보안**: CSRF, XSS, SQL Injection 방어, 세션 보안

## 자주 묻는 질문 (FAQ)

**Q: 전문 지식이 없어도 운영할 수 있나요?**
A: 네. 관리자 화면은 직관적으로 설계되어 있어, 블로그를 쓸 수 있는 정도면 충분합니다. 납품 시 사용법 안내도 포함됩니다.

**Q: 기존 사이트에서 이전할 수 있나요?**
A: 가능합니다. 기존 콘텐츠, 회원 데이터 등의 마이그레이션도 대응합니다. (별도 견적)

**Q: 도메인이나 서버는 어떻게 하나요?**
A: 기존 도메인/서버가 있으면 그대로 사용 가능합니다. 없으시면 추천 및 설정 대행도 해 드립니다.

**Q: 납품 후 수정이 필요하면?**
A: 납품 후 3개월간 불량은 무상 대응합니다. 기능 추가나 변경은 별도 견적으로 대응합니다.

**Q: 일본어 외에 다른 언어도 지원하나요?**
A: VosCMS는 한국어, 영어, 일본어, 중국어(간체/번체), 독일어, 프랑스어, 스페인어, 인도네시아어, 몽골어, 러시아어, 터키어, 베트남어 — 총 13개국어를 지원합니다.

**Q: 월 유지비가 있나요?**
A: VosCMS 자체에는 월 과금이 없습니다. 서버 비용(월 ¥1,000~)만 별도입니다.
