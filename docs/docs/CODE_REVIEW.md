## 2. 코드 품질

| 항목 | 평가 |
|------|------|
| SQL Injection | **양호** - prepare/bindParam 사용, 직접 쿼리 삽입 없음 |
| XSS | **주의** - 뷰 20개 파일에서 `$_GET/$_POST` 직접 접근 98건. 이스케이프 누락 가능 |
| CSRF | **양호** - CsrfMiddleware, POST 라우트에 적용 |
| 에러 처리 | **보통** - ValidationException, Logger, 403/404 페이지 존재 |
| 코드 중복 | **보통** - 공용 컴포넌트 잘 분리됨 |
