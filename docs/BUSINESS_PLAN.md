## 4. 매장 형태별 구조

### 4.1 개인 살롱 (현재)

```
RezlyX Salon
└── 1개 매장
```

### 4.2 멀티 브랜치 (단일 브랜드, 복수 매장)

```
RezlyX Salon Group
└── EARTH (브랜드)
    ├── 시부야점 (Branch)
    ├── 신주쿠점 (Branch)
    └── 이케부쿠로점 (Branch)
```

- 같은 브랜드, 같은 서비스 메뉴
- 매장별로 스태프/스케줄/예약만 분리

### 4.3 살롱 그룹 (멀티 브랜드)

```
RezlyX Salon Group
└── AFLOAT Group
    ├── AFLOAT D'L (럭셔리 컨셉)
    │   ├── 아오야마점
    │   └── 긴자점
    ├── AFLOAT RUVUA (캐주얼 컨셉)
    │   └── 시부야점
    └── AFLOAT WORLD (글로벌 컨셉)
        └── 하라주쿠점
```

- 브랜드마다 컨셉/서비스/가격이 다름
- 각 브랜드 아래에 매장(브랜치)

## 5. 기술적 구조

- 코드베이스 1개, 라이선스로 기능 분기

- 멀티 브랜치와 그룹의 기술 차이: brands 테이블 1개 + UI 1단계 추가뿐

```

  → 2년 합계: 패키지 ¥1,140,000 vs SaaS ¥3,600,000
```

### 8.3 디자이너 포인트 시스템

| 활동 | 보상 |
|------|------|
| 헤어스토리 작성 | +포인트 |
| Before/After 등록 | +포인트 |
| 고객 리뷰 획득 | +포인트 |
| 예약 성사 | +포인트 |
| 스타일북 좋아요 | +포인트 |

**환전:** 포인트 → 현금 (계좌 이체), 최소 환전 금액 설정

**비즈니스 효과:**
- 디자이너: ¥5,000 투자 → 활동하면 회수 가능, PR + 수익
- 플랫폼: 디자이너가 자발적으로 양질의 콘텐츠 생산 → 고객 유입 → 매장 유료 전환
- 매장: 소속 디자이너의 활동 → 매장 홍보 효과, 부담 없음
- 등록비 ¥5,000이 콘텐츠 엔진의 시드 머니 역할

### 8.4 예약 수수료

**건당 ¥200 (시술금액 무관, 고정)**

#### 구조

```
고객이 예약 시 ¥1,000 결제 (예약금/デポジット)
  → 전액 시술비에서 차감 (고객 추가 부담 ¥0)

매장 정산:
  시술비 ¥10,000
  - 예약금 ¥1,000 (고객 이미 결제)
  - 수수료 ¥200
  = 당일 결제 ¥9,000 + 예약금 정산 ¥800
  → 매장 실수령: ¥9,800

노쇼 시: 예약금 ¥1,000 몰수 (매장 ¥800 + 플랫폼 ¥200)
```

| ¥1,500,000 (200건) | ¥80,000 | ¥105,000 | **HPB보다 비쌈** |
| ¥3,000,000 (400건) | ¥110,000 | ¥205,000 | **HPB의 2배** |

→ ¥500은 중~대규모 매장에서 HPB보다 비싸져 경쟁력 상실.
→ **¥200은 모든 규모에서 HPB 대비 23~65% 저렴** — 확실한 셀링 포인트.

### 8.5 패키지 vs SaaS 포지셔닝

| | 패키지 (Salon/Group) | SaaS (rezlyx.com) |
|---|---|---|
| **타겟** | 자체 서버 운영, 브랜드 독립 | 간편하게 시작, 인프라 부담 없음 |
| **장점** | 소유권, 커스터마이징, 장기 비용 유리 | 초기 비용 ¥0, 즉시 시작 |
| **수익** | 1회 라이선스 + 유지보수 | 월정액 반복 수익 |
| **고객 흐름** | SaaS 무료 → 성장 → 패키지 전환 가능 | 포털에서 유입 |

### 8.6 로드맵

1. rezlyx.com 포털 — 매장 등록/검색/홍보 (현재 RezlyX에 vos-shop 플러그인)
2. 온라인 예약 SaaS — 포털 내 예약 기능 활성화
3. 운영 도구 SaaS — POS/키오스크/모바일 접수 클라우드 제공
4. 디자이너 페이지 — 개인 포트폴리오/헤어스토리

## 9. 전략 아이디어 (논의 예정)

### 9.1 디자이너 포트폴리오 개인 귀속

디자이너 이직/독립 시 헤어스토리, 포트폴리오, 포인트가 **개인 계정에 귀속**. 매장을 옮겨도 포트폴리오 유지 → 디자이너가 rezlyx.com을 떠나지 않는 구조. Hot Pepper에서는 불가능한 차별점.

### 9.2 디자이너 기반 역방향 예약 흐름

기존: 매장 검색 → 스태프 선택 → 예약
추가: **헤어스토리 브라우징 → 디자이너 발견 → 매장 확인 → 지명 예약**
Instagram에서 디자이너를 발견하는 경험을 rezlyx.com 안에서 제공.

### 9.3 디자이너 등급 시스템

| 등급 | 조건 | 혜택 |
|------|------|------|
| Bronze | 등록 시 | 기본 페이지 |
| Silver | 헤어스토리 10개 + 리뷰 20개 | 상위 노출, 배지 |
| Gold | 헤어스토리 30개 + 예약 100건 | 추천 디자이너 표시 |
| Platinum | 상위 1% | 메인 페이지 피처링 |

등급 상승 → 노출 증가 → 예약 증가 → 포인트 증가의 선순환.

### 9.4 무료 매장 대량 등록 전략

무료 매장 수 = 플랫폼 가치. 공개 정보로 기본 페이지를 생성하고 "이건 당신의 매장입니다, 관리하시겠습니까?"로 접근 (Google My Business 방식). 무료 매장 1,000개 → 포털로 자리잡음 → Google 검색 노출 → 자연 유입.

### 9.5 예약 수수료 ¥0 마케팅

Hot Pepper Beauty 최대 불만인 예약 건당 수수료를 정면 공략.
"Hot Pepper 수수료 매달 ¥50,000+ vs rezlyx.com 월 ¥5,000 정액, 수수료 ¥0"
이 한 마디로 영업 가능.

### 9.6 패키지 ↔ SaaS 상호 보완

패키지 구매 고객도 rezlyx.com 포털에 무료 등록 → 추가 유입 채널.
SaaS → 성장 시 패키지 전환 인센티브 (기존 납부액 라이선스에 충당).
경쟁이 아닌 보완 구조.

### 9.7 장기 비전

```
1단계: 패키지 판매 (즉시 수익)
2단계: SaaS 포털 (반복 수익)
3단계: 플랫폼 (네트워크 효과 — 매장↔고객↔디자이너)
4단계: 데이터 비즈니스 (지역별 트렌드, 인기 스타일, 가격 분석)
```
