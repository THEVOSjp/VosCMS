# 다국어 시스템 (I18n)

RezlyX의 다국어(국제화) 시스템 문서입니다.

## 지원 언어

| 코드 | 언어 | 이름 |
|------|------|------|
| `ko` | 한국어 | 한국어 |
| `en` | English | 영어 |
| `ja` | 日本語 | 일본어 |

---

## 사용법

### 초기화

```php
use RzxLib\Core\I18n\Translator;

// 번역 파일 경로와 함께 초기화
Translator::init(__DIR__ . '/resources/lang');

// 특정 로케일로 초기화
Translator::init(__DIR__ . '/resources/lang', 'ko');
```

### 번역 가져오기

```php
// 헬퍼 함수 사용
echo __('common.nav.home');           // "홈"
echo __('booking.title');              // "예약하기"

// 변수 치환
echo __('common.pagination.showing', [
    'from' => 1,
    'to' => 10,
    'total' => 100
]);  // "100개 중 1 - 10 표시"

// 마이페이지 환영 메시지
echo __('mypage.welcome', ['name' => '홍길동']);  // "홍길동님, 환영합니다!"
```

### 로케일 변경

```php
use RzxLib\Core\I18n\Translator;

// 현재 로케일 확인
echo Translator::getLocale();  // "ko"

// 로케일 변경 (세션/쿠키에 저장됨)
Translator::setLocale('en');

// 지원 로케일 목록
$locales = Translator::getSupportedLocales();  // ['ko', 'en', 'ja']

// 로케일 정보 (드롭다운용)
$allLocales = Translator::getAllLocales();
// [
//     ['code' => 'ko', 'name' => '한국어', 'active' => true],
//     ['code' => 'en', 'name' => 'English', 'active' => false],
//     ['code' => 'ja', 'name' => '日本語', 'active' => false],
// ]
```

### 복수형 처리

```php
// 번역 파일에서
// 'items' => '아이템 없음|:count개 아이템|:count개 아이템들'

echo trans_choice('common.items', 0);   // "아이템 없음"
echo trans_choice('common.items', 1);   // "1개 아이템"
echo trans_choice('common.items', 5);   // "5개 아이템들"
```

## 헬퍼 함수

| 함수 | 설명 |
|------|------|
| `__($key, $replace)` | 번역 가져오기 |
| `trans($key, $replace)` | `__`의 별칭 |
| `trans_choice($key, $count, $replace)` | 복수형 번역 |
| `current_locale()` | 현재 로케일 |

## 번역 키 규칙

### 네이밍 컨벤션

```
{파일}.{섹션}.{키}

common.nav.home       # 공통 > 네비게이션 > 홈
booking.form.email    # 예약 > 폼 > 이메일
admin.services.title  # 관리자 > 서비스 > 제목
```

### 파일별 역할

| 파일 | 내용 |
|------|------|
| `common.php` | 네비게이션, 버튼, 상태, 시간, 공통 메시지 |
| `booking.php` | 예약 폼, 확인, 완료, 취소 관련 |
| `services.php` | 서비스 목록, 상세, CTA |
| `mypage.php` | 마이페이지, 예약 내역, 프로필 |
| `auth.php` | 로그인, 회원가입, 비밀번호 |
| `admin.php` | 관리자 대시보드, CRUD 작업 |
| `validation.php` | 유효성 검사 에러 메시지 |

## 로케일 자동 감지

시스템은 다음 순서로 로케일을 감지합니다:

1. **세션** (`$_SESSION['locale']`)
2. **쿠키** (`$_COOKIE['locale']`)
3. **브라우저** (`Accept-Language` 헤더)
4. **기본값** (`ko`)

## DB 기반 다국어 번역

파일 기반 번역 외에 DB에 저장되는 사용자 정의 번역을 지원합니다.

### 테이블 구조

```sql
CREATE TABLE `rzx_translations` (
    `id` INT AUTO_INCREMENT NOT NULL,
    `lang_key` VARCHAR(255) NOT NULL,    -- 번역 키 (예: 'site.name')
    `locale` VARCHAR(10) NOT NULL,        -- 언어 코드 (ko, en, ja)
    `content` TEXT NOT NULL,              -- 번역 내용
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_key_locale` (`lang_key`, `locale`)
);
```

### 헬퍼 함수

| 함수 | 설명 |
|------|------|
| `db_trans($key, $locale, $default)` | DB에서 번역 가져오기 |
| `get_site_name($locale)` | 사이트 이름 (다국어) |
| `get_site_tagline($locale)` | 사이트 제목 (다국어) |
| `get_setting($key, $default)` | 설정값 가져오기 |

### 사용법

```php
// DB 번역 가져오기
$tagline = db_trans('site.tagline', 'ko', '기본값');

// 사이트 이름 (우선순위: DB 번역 > 설정 > 환경변수)
$siteName = get_site_name();

// 사이트 제목 (우선순위: DB 번역 > 설정)
$siteTagline = get_site_tagline();
```

### 우선순위

1. `rzx_translations` 테이블 (다국어 모달에서 입력한 값)
2. `rzx_settings` 테이블 (입력창에 직접 입력한 값 - 모든 언어 공통)
3. 환경변수 또는 기본값

### 번역 키 예시

| 키 | 설명 |
|----|------|
| `site.name` | 사이트 이름 |
| `site.tagline` | 사이트 제목/슬로건 |
| `service.name.{id}` | 서비스 이름 |
| `service.description.{id}` | 서비스 설명 |


## 네이티브 국가명 (Endonym) 시스템

전화번호 입력 컴포넌트에서는 번역 파일 대신 네이티브 국가명을 사용합니다.

### 개념

각 국가명을 해당 국가의 공식 언어로 표시하여 번역 파일 유지보수 부담을 제거합니다.

| 국가 | 네이티브명 | 설명 |
|------|-----------|------|
| 🇰🇷 한국 | 대한민국 | 한국어 |
| 🇯🇵 일본 | 日本 | 일본어 |
| 🇩🇪 독일 | Deutschland | 독일어 |
| 🇪🇸 스페인 | España | 스페인어 |
| 🇷🇺 러시아 | Россия | 러시아어 |
| 🇸🇦 사우디 | السعودية | 아랍어 |
| 🇮🇱 이스라엘 | Israel | 히브리어/영어 |

### 구현 방식

```php
// resources/views/components/phone-input.php
$countries = [
    ['+82', 'kr', '대한민국'],
    ['+81', 'jp', '日本'],
    ['+49', 'de', 'Deutschland'],
    // ...
];

// 사용
foreach ($countries as $country) {
    $code = $country[0];     // '+82'
    $iso = $country[1];      // 'kr'
    $name = $country[2];     // '대한민국'
}
```

### 장점

- **번역 불필요**: 180개+ 국가의 번역 파일 유지보수 부담 없음
- **정확성**: 각 국가의 공식 명칭 사용
- **인식성**: 국기 이모지와 함께 표시되어 낯선 문자도 식별 가능
- **일관성**: 사용자 언어 설정과 무관하게 동일한 표시

### 기존 번역 키와의 관계

`common.countries.*` 키는 레거시 호환성을 위해 유지되지만, 전화번호 컴포넌트에서는 사용하지 않습니다.
