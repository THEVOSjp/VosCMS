## 1. 예약 시스템 위젯

| 설정 | 설명 |
|------|------|
| Title | 섹션 제목 (다국어) |

| Show Duration | 소요시간 표시 여부 |

스태프 목록을 카드 형태로 표시. 포지션 필터, 지명비, 지명예약 버튼 포함.

| 설정 | 설명 |
|------|------|
| Title / Subtitle | 섹션 제목/부제 (다국어) |
| Columns | 컬럼 수 (2~6) |
| Show Position Filter | 포지션 필터 탭 표시 |

| Show Booking Button | 지명예약 버튼 표시 |
| Show Services | 담당 서비스 표시 |
| Max Staff | 최대 표시 인원 (0=전체) |

### lookup — 예약 조회

전화번호 또는 예약번호로 예약 검색.

|------|------|
| Title | 섹션 제목 (다국어) |

### services — 서비스 목록

서비스를 카드 형태로 표시. 카테고리 필터 포함.

## 2. 콘텐츠 위젯

### before-after — Before / After

시술 전후 비교 사진. 나란히 보기 ↔ 드래그 비교 슬라이더 전환.

| 설정 | 설명 |
|------|------|
| Title | 섹션 제목 (다국어) |
| Source | 데이터 소스 (stylebook: DB 자동 / manual: 수동 입력) |
| Items | 수동 입력 시 Before/After 이미지 세트 |
| Auto Limit | 자동 표시 개수 (3/5/8) |
| Style | 슬라이더 / 그리드 |

**사용법:**
- `source: stylebook` → `rzx_before_afters` 테이블에서 자동 로드
- `source: manual` → Items에 직접 이미지 URL 입력
- 비교 버튼 클릭 → 드래그 슬라이더로 전환

### marketplace-showcase — 자료실 (v0.1.0)

마켓플레이스(`rzx_mp_items`)의 최신 아이템을 **가로 슬라이드 카드** 형태로 표시합니다.
좌우 화살표 버튼 또는 터치/마우스 드래그로 스크롤합니다.

**주요 기능:**
- DB에서 최신/인기/평점순으로 아이템 자동 조회
- 유형별 필터 (전체/모듈/테마/위젯/스킨)
- 카드 구성: 썸네일, 유형 배지, 가격/판매 배지, 제목, 설명, 저자, 평점, 다운로드 수
- CSS `scroll-snap` 기반 부드러운 스크롤 + JS 좌우 네비게이션
- 모바일 터치 스크롤 지원
- 다크모드 완전 대응
- 다국어 13개 언어 (설정 라벨, select 옵션 포함)

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 섹션 제목 | 상단 표시 제목 (다국어) | 자료실 |
| 표시 개수 | 카드 표시 수 (1~20) | 10 |
| 유형 필터 | 전체 / 모듈 / 테마 / 위젯 / 스킨 | 전체 |
| 정렬 | 최신순 / 인기순 / 평점순 | 최신순 |
| 더보기 링크 | 마켓플레이스 링크 표시 여부 | ON |
| 배경색 | 섹션 배경색 | 투명 |

**사용법:**
1. 위젯 빌더에서 **콘텐츠** 카테고리 → `자료실` 위젯 추가
2. 설정 패널에서 표시 개수, 정렬 방식, 유형 필터 설정
3. 섹션 제목은 다국어 입력 가능 (🌐 버튼)
4. [적용] → [레이아웃 저장]

**데이터 소스:** `rzx_mp_items` 테이블 (vos-marketplace 플러그인 필요)

**파일 구조:**
```
widgets/marketplace-showcase/
├── widget.json   — 위젯 설정 스키마 (13개 언어)
└── render.php    — PHP 렌더러 (DB 조회 + HTML/CSS/JS 생성)
```

### board-showcase — 쇼케이스 (v0.1.0)

게시판의 최신 글을 **가로 슬라이드 카드** 형태로 표시합니다.
좌우 화살표 버튼 또는 터치/마우스 드래그로 스크롤합니다.

**주요 기능:**
- 위젯 빌더에서 게시판을 선택 (board_select 드롭다운)
- 게시물 수 설정 가능 (1~30개)
- 카드 구성: 썸네일(첨부 이미지 또는 본문 img 추출) + 제목 + 설명(80자) + 저자 + 날짜 + 조회수
- CSS `scroll-snap` 기반 부드러운 스크롤 + JS 좌우 네비게이션
- 모바일 터치 스크롤 지원
- 다크모드 완전 대응
- 다국어 13개 언어 (설정 라벨 포함)
- `db_trans_batch()` 지원 (게시글 다국어 번역)

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 섹션 제목 | 상단 표시 제목 (다국어) | 쇼케이스 |
| 게시판 선택 | 표시할 게시판 (드롭다운) | — |
| 게시물 수 | 카드 표시 수 (1~30) | 10 |
| 더보기 링크 | 게시판 목록 링크 표시 여부 | ON |
| 배경색 | 섹션 배경색 | 투명 |

**사용법:**
1. 위젯 빌더에서 **콘텐츠** 카테고리 → `쇼케이스` 위젯 추가
2. 설정 패널에서 게시판 선택 (notice, free 등)
3. 게시물 수 조정
4. 섹션 제목은 다국어 입력 가능 (🌐 버튼)
5. [적용] → [레이아웃 저장]

**데이터 소스:** `rzx_board_posts` + `rzx_board_files` (첨부 이미지)

**파일 구조:**
```
widgets/board-showcase/
├── widget.json   — 위젯 설정 스키마 (13개 언어)
└── render.php    — PHP 렌더러 (DB 조회 + HTML/CSS/JS 생성)
```

### download-hero — 다운로드 히어로 (v1.0.0)

| 항목 | 내용 |

**섹션**:
- 히어로 배너 (버전 배지 + 다운로드/변경이력/GitHub 버튼)

### contact-form — 문의하기 (v0.1.0)

비공개 1:1 문의 폼. 비회원도 이용 가능하며, DB 저장 + SMTP 메일 알림을 지원합니다.
위젯 타입이므로 어떤 페이지에든 배치할 수 있습니다.

**주요 기능:**
- 이름, 이메일, 분류, 제목, 내용 입력 폼
- 로그인 시 이름/이메일 자동 입력
- 분류 선택: 일반 문의 / 사업 제안 / 라이선스 / 버그 리포트 / 보안 취약점 / 기타
- 허니팟 스팸 방지
- AJAX 전송 + 성공/실패 피드백
- `rzx_contact_messages` 테이블에 저장
- `rzx_send_mail()` 공용 헬퍼로 관리자 이메일 알림 (SMTP)
- 13개 언어 폼 레이블/메시지/플레이스홀더
- 개인정보 안내 문구 표시
- 다크모드 완전 대응

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 섹션 제목 | 상단 표시 제목 (다국어) | 문의하기 |
| 부제 | 안내 문구 (다국어) | 궁금한 점이 있으시면... |
| 분류 선택 표시 | 카테고리 드롭다운 표시 여부 | ON |
| 수신 이메일 | 문의 알림 받을 이메일 | — |
| 배경색 | 섹션 배경색 | 투명 |

**사용법:**
1. 위젯 빌더에서 **시스템** 카테고리 → `문의하기` 위젯 추가
2. 설정에서 수신 이메일 입력 (SMTP 메일 알림용)
3. [적용] → [레이아웃 저장]

**데이터 소스:** `rzx_contact_messages` 테이블
**메일 발송:** `rzx_send_mail()` — DB SMTP 설정 자동 사용

**파일 구조:**
```
widgets/contact-form/
├── widget.json   — 위젯 설정 스키마 (13개 언어)
└── render.php    — PHP 렌더러 (폼 + API + 메일 발송)
```

### feature-tour — 기능 소개 투어

스크린샷 슬라이드 + 다크 패널 기능 소개. Product Tour 스타일.

| 설정 | 설명 |
|------|------|

| Panel Main Copy | 우측 패널 메인 카피 (다국어, 줄바꿈 가능) |
| Panel Description | 우측 패널 설명 (다국어) |
| Bullet Points | 우측 패널 불릿 포인트 (줄바꿈으로 구분, 다국어) |
| Feature Slides | 슬라이드 아이템 (이미지/동영상 + 제목 + 설명) |

**슬라이드 입력 방법:**
1. 편집 패널에서 "Feature Slides" 섹션의 [Add] 클릭
2. 이미지 또는 동영상 업로드
3. 제목, 설명 입력 (다국어 지원)
4. 필요한 만큼 반복 추가
5. 적용 → 프리뷰

**동작:**
- 좌측 스크린샷 카드: fade 전환 슬라이드
- 우측 다크 패널: 프로그레스 바 + 좌우 버튼 + 도트 인디케이터
- 자동 슬라이드 5초 (호버 시 정지)
- 모바일 스와이프 지원
- **이미지 클릭 → 라이트박스 (원본 슬라이드 뷰)**

**라이트박스:**
- 클릭 → 현재 슬라이드 위치에서 열림
- 좌우 화살표 → 슬라이드 전환
- 키보드 ←→ 화살표, ESC 닫기
- 모바일 스와이프
- 동영상 → controls 표시, 자동재생
- 닫기 → X 버튼, 배경 클릭, ESC
- 하단 → 기능명 + 설명 표시
- 좌상단 → 카운터 (1/7)

### stats — 통계 카운터 (v1.2.0)

숫자 통계 카드 + 카운트업 애니메이션.

**아이템 필드**
- `number` — 숫자 (예: `10000`) 또는 문자열 (`24/7`). 순수 숫자만 카운트업 대상
- `suffix` — 접미사 (예: `+`, `%`)
- `label` — 라벨 (다국어 객체)
- `icon` — 16종 중 선택 (users, chart, star, heart, check-circle, clock, calendar, globe, shield, lightning, trophy, cart, dollar, building, chat) · 생략 시 아이콘 없음
- `color` — 10종 (blue, green, purple, red, orange, indigo, pink, teal, amber, zinc)

**전역 옵션**
- `animate` — 카운트업 애니메이션 on/off (기본 on)

**동작**
- 관리자 빌더에서 아이템 추가/수정/삭제, 순서는 배열 순서
- 뷰포트 진입 시 IntersectionObserver 가 `.rzx-stat-item` 별로 0 → 목표값 애니메이션 (1.4s, easeOutCubic)
- `data-stat-delay` 로 100ms 씩 계단식 지연
- 콤마 서식·소수점 유지, IO 미지원 환경 폴백 지원

## 위젯 배치 방법

1. 관리자 → 페이지 관리 → 원하는 페이지의 [수정] 클릭
2. 좌측 위젯 팔레트에서 카테고리 선택
3. 위젯 클릭 → 캔버스에 추가
4. 위젯 클릭 → 우측 편집 패널에서 설정
5. [적용] → 프리뷰 확인
6. [저장] → 배포
